## Key Lessons From My Analytical Errors

### Error 1: Reading intraday fade as multi-day reversal (4/23)

**What went wrong**:
- Anchored on textbook "exhaustion gap + fade" pattern
- Underweighted the magnitude of the guidance raise (doubled growth assumption)
- Ignored sector co-rally (CIEN, LITE, COHR also up that week)
- Ignored my own strategies.md rule about post-earnings drift continuing 3-5 days
- Gave specific probability (60-70% for fade) on pattern-recognition without flow confirmation

**What should have happened**:
- Run the 4-factor confirmation check from pitfall 20
- 3 of 4 bullish (fundamentals confirmed, sector co-moving, KOL narrative spreading; only flow was ambiguous on day-of) → DO NOT predict fade
- The strategies.md "post-earnings drift T+1 to T+5" rule should have outranked the intraday pattern
- Should have written: "Today's intraday fade is one-session profit-taking. Default expectation is continuation 3-5 days. Reload zones $10.30-10.50 are plausible but not required for entry."

### Error 2: Calling IV crush without flow check (5/11)

**What went wrong**:
- Treated IV Rank 66 as "near peak, due to crush" without checking what was *causing* the IV
- Failed to recognize Q2 earnings is 2 months away — no near-term event for the IV to crush off
- Did not pull net premium / large block data BEFORE making the recommendation
- Conflated event-driven IV (pre-earnings) with demand-driven IV (sustained accumulation)

**What should have happened**:
- Catalyst clock check: next earnings ≥ 30 days = not event-IV
- Flow data pull: net call premium +$14M, multiple multi-hundred-K block buys, OTM expirations being bought aggressively = demand-IV
- Term structure check: back-month LEAPS IV also elevated → demand-IV signature
- Conclusion should have been: "IV elevation is real options demand, not pre-event premium. Holding long-vol exposure is correctly aligned. Do not roll until flow reverses."

### Meta-insight

Both errors share a common root: **applying a generic pattern-recognition heuristic without first pulling the data that would falsify it**. The patterns ("gap-fade = multi-day top", "high IV = crush coming") are real priors, but updated probabilities require **flow + catalyst-clock + sector** evidence each time.

**The user's pushbacks were data-driven**; my predictions were pattern-driven. Pattern beats data only when data is unavailable. Here it was always available — I just didn't pull it before publishing the recommendation.

## Reusable Framework

### Pre-event vs post-event analysis order

**Before earnings**:
1. Flow data → who's accumulating into the print?
2. IV term structure → event-driven peak or sustained demand?
3. Catalyst clock → what's the next event after this one?

**Post-event T+0**:
1. Did fundamentals confirm thesis? (binary)
2. Did guidance raise? (key signal)
3. Sector reaction (co-move = thematic, isolated = stock-specific)
4. Intraday tape: distribution OR rotation in? (volume + closing price relative to VWAP)

**T+1 to T+5 (drift window)**:
1. Flow data daily (net premium direction)
2. Sector continuation
3. Sell-side actions (upgrades cascading?)
4. Closing trend (each day close above prior close = momentum intact)

### When to update conviction higher post-earnings

If all 4 hold by T+2:
- Fundamentals confirmed (decisive beat + raise)
- Sector co-moving (peers up >5% same period)
- Net call premium >+$5M sustained
- Closing above T+0 high

→ Continuation likely 1-3 more weeks. Hold or add. **Do not predict reversal**.

### When to flip thesis bearish

- Beat but guidance LOWERED (e.g., one-time benefit but next quarter weakness)
- Sector breaking down same week
- Net premium turning negative on T+1 (institutional distribution)
- Stock closes below T+0 low on T+2
